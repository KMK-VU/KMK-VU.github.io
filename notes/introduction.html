JS is love , JS is life <3

Static website for custom javascript experiments.

The purpose of this website is to test some javascript codes that run solely on client side , a.k.a your computer. The idea is to explore what can be done using only frontend capabilities. But why?

  • Because it's free ! No need to pay for a server to do computations and there are free hosting solutions available for simple static websites like github pages.
  • Because it's ubiquitous ! Everyone has a web browser to run and interact with your code on internet, and even without internet by just saving files.
  • Because it's lightweight ! Fire up a text editor, write some js code, add a pinch of html et voila ! You can host it or just send the files. No need of programming environments on dev or user side. Just a web browser.
  • Because why not ? Let's have some fun with programming from time to time. No need to always look for the perfect solution especially when it's not at work.
Sceptical about webassembly solutions but maybe one day...

The different js experiments showcased here are currently divided into 3 categories: Algorithms, Data and Games.

Experiments are primarily designed to work on desktop. Some experiments might not work on mobile.

This website does not have the pretention to explain or teach you anything about programming, javascript or the different topics involved. It's just about tests & interactions.

Therefore don't expect production quality codes here. But if you have any feedbacks on how to improve the experiments, I am interested.
